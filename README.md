# ProjectHub — College Project Management System

Full-stack MERN app with three roles: Admin, Faculty, Student.

## AI setup

Set one provider key in `backend/.env` to enable live AI recommendations, project summaries, and code reviews:

- `OPENAI_API_KEY` with optional `OPENAI_MODEL`
- `GEMINI_API_KEY` with optional `GEMINI_MODEL`

If no key is configured, ProjectHub keeps working with local fallback suggestions and summaries.
